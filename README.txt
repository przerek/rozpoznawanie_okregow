Program wykrywający drugi wewnętrzny okrąg. Napisany w języku java. Do uruchomienia wymagana jest biblioteka OpenCV wraz z opcją uruchomieniową(VM option), np.
-Djava.library.path="C:/opencv/build/java/x64"

Efekty dzialania programu dla różnych danych wejściowych zaprezentowano w folderze demonstration.
